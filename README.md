# 📦 Inventario de Útiles Escolares

Aplicación web desarrollada en Python para gestionar un inventario de útiles escolares. Permite visualizar, editar y administrar productos de forma sencilla.

---

## 🚀 Tecnologías utilizadas

* Python 🐍
* Flask
* HTML5
* CSS

---

## 📂 Estructura del proyecto

```
inventario-utiles-escolares/
├── app.py
├── templates/
│   ├── index.html
│   └── editar.html
```

---

## ⚙️ Instalación y ejecución

1. Clona el repositorio:

```
git clone https://github.com/shammysuarez-ui/inventario-utiles-escolares.git
```

2. Entra a la carpeta del proyecto:

```
cd inventario-utiles-escolares
```

3. Instala las dependencias:

```
pip install flask
```

4. Ejecuta la aplicación:

```
python app.py
```

5. Abre tu navegador en:

```
http://127.0.0.1:5000/
```
